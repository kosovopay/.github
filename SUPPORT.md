# Getting support

Thanks for using KosovoPay. Here's the fastest path to an answer.

| You want to… | Go to |
|---|---|
| Read the API reference + guides | **https://pay.kosovo.sh/docs** |
| Report a bug in an SDK | Open an issue in that SDK's repo |
| Request a feature | Open a feature-request issue |
| Ask an integration question | **developers@kosovo.sh** |
| Report a security issue | **security@kosovo.sh** (see [SECURITY](SECURITY.md)) |
| Account / billing / live-key access | **support@kosovo.sh** |

**Before opening an issue:** check the [docs](https://pay.kosovo.sh/docs) and existing
issues. When you do open one, include your SDK version, language runtime, and a minimal
reproduction — it gets you an answer far faster.

> GitHub issues are for **bugs and feature requests**, not private account matters. For
> anything involving keys, money, or personal data, email us instead.
